# Hyped Cards
Turn Hype bracelets to some cool-looking RFID cards using a ESP32

## How it should work
Hype bracelets use RFID to store a ID like 'A3 7F 92 1B' witch is just the identification of the bracelet.
So if we give the same ID to a random RFID card it should work as the bracelet.
